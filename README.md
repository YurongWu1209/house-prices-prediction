# House Prices Prediction

Kaggle competition:
House Prices - Advanced Regression Techniques

## Goal

Predict house sale prices using feature engineering and machine learning models.

## Techniques Used

- Data Cleaning
- Missing Value Imputation
- Outlier Removal
- Log Transformation
- One-Hot Encoding
- Ordinal Encoding
- Feature Engineering
- XGBoost
- Cross Validation
- Early Stopping

## Feature Engineering

Created additional features:

- TotalArea
- TotalBathrooms
- TotalPorchArea
- HouseAge
- RemodAge
- HasGarage
- HasBsmt

## Model

XGBoost Regressor

## Results

Initial score:

0.13969

Final score:

0.13230

## Lessons Learned

- Proper ordinal encoding matters.
- Feature engineering can improve performance significantly.
- Cross-validation helps estimate leaderboard performance.
